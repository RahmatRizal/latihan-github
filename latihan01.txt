GitHub adalah layanan hos web bersama untuk proyek pengembangan perangkat lunak yang menggunakan sistem kendali versi Git dan layanan hosting internet. Hal ini banyak digunakan untuk kode komputer. Wikipedia

Github berbeda dengan Git. Git adalah perangkat lunak yang digunakan lokal pada komputer anda di rumah sedangkan Github adalah layanan cloud untuk membuat repository yang disediakan oleh website Github.

VCS / Version Control System adalah sebuah sistem untuk mengelola perubahan pada file, dokumen, atau source code anda
